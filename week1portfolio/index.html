Flexbox & Grid Layout

1. Difference Between Flexbox and Grid

Flexbox is a one-dimensional layout system used to arrange elements in a row or a column. It is best for simple alignments like navbars or single-direction content.

Grid is a two-dimensional system that works with both rows and columns. It is ideal for complex layouts like galleries or project grids.

2. My Choice

I used CSS Grid for my Projects section and Skills section because it allows me to structure sections in rows and columns easily and makes my layout responsive on all screen sizes.

3. Code Implementation

Projects Section Grid:


.projects-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 20px;
}

Skills Section Grid:


.skills-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
}
